Sincronizar
Importar e exportar dados via Google Drive ou arquivo
☁️ Google Drive — arquivo JSON compartilhado
ℹ️ Para sincronizar via Google Drive, você precisa publicar o arquivo JSON como link público e colar o ID do arquivo ou a URL de exportação abaixo. Todos os usuários usarão o mesmo link para carregar os dados.
Cole o link e clique em Carregar.
⚙️ Como obter o link correto do Google Drive:
No Google Drive, clique com botão direito no arquivo .json → Compartilhar → Qualquer pessoa com o link → Copiar link.
O link parecerá com: https://drive.google.com/file/d/ID/view — copie apenas o ID (parte em negrito).
Cole o ID no campo acima. O sistema monta automaticamente a URL de download direto.
Clique em Carregar. Se der erro de CORS, use a opção de importar arquivo abaixo.
📥 Importar JSON
Carregue um arquivo .json exportado anteriormente pelo sistema. Os dados serão substituídos pelos do arquivo.
📂
Clique para selecionar ou arraste o arquivo aqui
📤 Exportar JSON
Salve todos os dados locais em um arquivo .json para enviar ao Google Drive ou compartilhar com outros usuários.
Após baixar, faça upload desse arquivo no Google Drive para que todos acessem a versão mais recente.
